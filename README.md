### 一、项目概述
**Hello** 👋<br/>
大家好：<br>
**`SkillHub·DouCY`** 是我的个人主页网站，它作为一个一站式快捷入口，不仅能让用户快速访问我的其他站点，还展示了我的个人技能和简介，是我在互联网上分享技术心得、项目经验以及个人成长历程的重要平台。

### 二、个人简介
**窦长友**，19岁，是一名热爱技术的学生。在编程和开源探索的道路上不断前行，对 Tech 和 Photograph 充满热情，目前正在尝试从事互联网相关工作。他坚信通过开源和知识共享，能与大家共同进步，因此创建了这个个人网站。

### 三、网站功能与特色
#### 1. 多功能板块整合
- **导航功能**：网站首页的导航栏提供了便捷的链接，涵盖“Home”、“关于我”、“联系我”、“旗下站点”、“友链”等板块，还有加入交流群的入口，方便用户快速找到所需内容。
- **展示平台**：“关于我”板块展示个人照片、简介和技能水平，“旗下站点”板块陈列多个相关网站，如记录学习笔记的“Smart Space”、文件存储平台“Peak Cloud Drive”等，“我的摄影作品”网站则展示摄影成果。“Blog”用于分享个人想法、经验和见解。
#### 2. 社交互动拓展
- **交流群入口**：“加入我们”链接指向飞书交流群，方便大家在线实时沟通，交流网站内容、技术问题或兴趣爱好相关话题。
- **多平台链接**：在“联系我”板块列出了多个平台链接，包括Github、Gitee、Blog、Mail、QQ、CSDN、小红书、500px等，拓展互动渠道，方便用户从不同平台了解我。
#### 3. 个人能力展示与版权声明
- **技能呈现**：在“关于我”板块，通过进度条形式展示在HTML5/CSS3、JavaScript/Typescript、PHP、数字设计等方面的能力，让访问者直观了解我技术水平。
- **版权保护**：网站底部明确版权信息“© 2024 - 2025 dcyyd_kcug@yeah.net, 窦长友 版权所有”，并附上版权协议链接，保护网站内容和个人创作成果。
#### 4. 友链推广与资源共享
- **友链推荐**：“友链”板块设置了“IntelliJ IDEA Blog”“牛牛技术客栈”等友链，为访问者提供更多有价值的外部资源链接，促进不同网站间的相互推广与合作。 

### 四、技术实现
#### 1. 前端技术
- **HTML5**：构建网站的基本结构，合理组织各个页面元素，形成清晰的页面层次。
- **CSS3**：运用丰富的样式属性，打造网站的视觉风格，包括响应式布局，确保在不同设备上都有良好的浏览体验，以及精心搭配字体、颜色等细节。
- **JavaScript**：为网站添加动态交互功能，如页面元素的动态效果、数据验证、异步加载等，提升用户与网站的互动性和操作便捷性。
#### 2. 后端技术
暂未涉及。
#### 3. 工具与框架
项目使用了一些工具和框架来辅助开发与部署，如Jekyll静态站点生成器，它简化了网站的构建和部署流程，通过灵活的模板系统，实现代码复用和网站结构的快速搭建，并能与GitHub Pages无缝集成，方便网站上线发布。同时，引入了如Font Awesome等外部库，用于增强页面的图标展示效果。
